Unicode Text Rendering Reference System

Unicode Text Rendering Reference System (UTRRS) is a open source web-based application to compare a rendered text character, rendered by a text rendering engine with reference image of a text character. Differences between the rendered image and reference image may be compared for the subsequent analysis.

Performance of a text rendering engine producing text according to typographical rules applicable to a natural language can be evaluated by one with no knowledge or ability to read the natural language. This Application Test for the Codepoints (Unicode Character set), GSUB(Glyph substitution) and GPOS (Glyph Positioning).

For the Indian Language present here, this system has been developed according to the IDN Script Grammar rules published by Govt. of India.

Languages Included :
  1. Assamese
  2. Bengali (India)
  3. Gujarati
  4. Hindi
  5. Kannada
  6. Malayalam
  7. Marathi
  8. Odia
  9. Punjabi
  10. Tamil
  11. Telugu

UTRRS Tool has been developed by Red Hat Language Technology Group. It is vastly used in various internal projects, localization support, Desktop Rendering, etc. in Red Hat.

UTRRS has been released by Red Hat, under open source MIT license in the year of 2010.

Usage :
  1. UTRRS provides the service to validate the standard shape (Rendering) of character set (Unicode 5.1 Standard) or composed characters (Syllable) produced in any system.
  2. This application can be used anywhere as a character archives for any Indian languages mentioned above, so far.
  3. UTRRS has been designed to facilitate the multilingual community users, print media, font developers & vendors, localized desktop providers, tourism industry, academic organizations and communities etc. in terms of providing them a standard ready reference for Indian languages.
